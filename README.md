# APS - Security Scan Dashboard

A React application for managing and monitoring security scans.

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Pages

1. **Login** - Sign up / Sign in page
2. **Dashboard** - View all scans, severity stats, search and filter
3. **Scan Detail** - Live scan console, activity logs, findings

## Folder Structure

```
src/
  pages/        - Login, Dashboard, ScanDetail
  components/   - Reusable UI and layout components
  context/      - Auth, Theme, Toast providers
  data/         - Mock data for scans, findings, logs
  lib/          - Utility functions
```

## Deployment

Deployed on Vercel.
